# Smoke Tracker

Мини-приложение для Telegram (Telegram Mini App) для контроля отказа от курения.

## Что делает
- Кнопка «Закурил» фиксирует время
- Живой таймер показывает, сколько времени прошло с последней сигареты
- Статистика за день и история записей
- Данные хранятся через Telegram CloudStorage (привязаны к аккаунту Telegram)

## Как подключить
1. Открыть GitHub Pages ссылку этого репозитория
2. В @BotFather: Bot Settings → Menu Button → указать ссылку

## Стек
Чистый HTML/CSS/JS, без бэкенда, без сборки.
